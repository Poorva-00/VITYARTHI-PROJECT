PROBLEM STATEMENT:

People often want a simple way to order food without manually calculating totals or remembering menus. 
This project solves that by providing a simple, automated Python system that accepts orders, calculates total bills, and simulates tracking.
This project aims to design a simple, user-friendly console-based Food Delivery System using Python.

The system allows users to:
* Enter their name and address
* View a menu
* Place an order
* Calculate bill
* Track delivery status

----

SCOPE OF THE PROJECT:

The scope of this project is to create a basic and easy-to-use food ordering system using simple Python. The system allows users to:


* View a menu containing food items, prices, and ingredients.

* Select items they want to order.

* Add multiple items to the order.

* Check the total bill, which is calculated automatically by the program.

* Receive order confirmation after placing the order.

* See simulated delivery status, such as preparing, picked up, and out for delivery

---

TARGET USERS:

This project aims to target customers of all age especially students and working professionals and also restaurant staffs and small food businesses
